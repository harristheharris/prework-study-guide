CSS

Terms

  • Margin: how much space we want outside an element
  • Padding: how much space do we want inside an element

Ways to style using CSS

  • Inline Styling
    • Stlyes an element directly
    • Applies style to specfic tag/s
    • Quick permanent changes but less flexible
    • Has highest priority
  • Internal Styling
    • Styles the HTML file directly under the < head > tag
    • Only applies style for that page/html file
    • A single change to the CSS will apply to all targeted tagged elements in that page/file
    • Mediuem priority
  • External Styling
    • Styles on an external file
    • The external file needs to be linked via the < link > tag and applies styling to all pages linked
    • This is the prefered method as it has the most flexibilty
    • Lowest priority

For Loops

A for loop contains 3 statements

  1. Defining variables
    • This is our starting point
    • e.g. var x = 0;
  2. The conditional
    • As long as this condtions remains true, the loop will contine
    • e.g x < shapes.length;
  3. Iteration
    • This is the engine that will repeat the process until the condition is not met
    • e.g. x++ or x+1

Full Example: for (var x = 0; x < shapes.length; x++) {}
