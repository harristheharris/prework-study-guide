Ways to style using CSS

  • Inline Styling
    • Stlyes an element directly
    • Applies style to specfic tag/s
    • Quick permanent changes but less flexible
    • Has highest priority
  • Internal Styling
    • Styles the HTML file directly under the < head > tag
    • Only applies style for that page/html file
    • A single change to the CSS will apply to all targeted tagged elements in that page/file
    • Mediuem priority
  • External Styling
    • Styles on an external file
    • The external file needs to be linked via the < link > tag and applies styling to all pages linked
    • This is the prefered method as it has the most flexibilty
    • Lowest priority
